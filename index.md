# The IB Library as the Academic Hub

In an IB school, the library is not merely a quiet space for books; it is an interdependent system of people, places, collections, and services [1]. This site is built around that idea — seven modules covering the skills, frameworks, and research an IB secondary librarian needs to run that system well across MYP and DP.

**Mission:** to function as the collaborative hub of the school's academic ecosystem, across both the Middle Years Programme and the Diploma Programme [2].

**Visual integration:** the physical and digital library spaces should reflect the **IB Learner Profile** — Inquirers, Knowledgeable, Open-Minded, and the rest — through themed book displays, reading challenges, and bulletin boards, embedding the library in the IB philosophy rather than treating it as a separate department [3].
