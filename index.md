Built for the day-to-day of supporting Grades 6–12 across MYP and DP. Everything lives in the navigation bar above — no homepage cards to click through, just a direct index of every page. Use the **Gap Research** and **Citations** dropdowns for the full list of subpages. Start wherever your current question is:

- **New to the role?** Start with the [Skills &amp; Resources Overview]({{ '/skills-overview/' | relative_url }}) and the [Research Skills Scaffolding Matrix]({{ '/scaffolding-matrix/' | relative_url }}) for how the librarian's job fits into MYP/DP.

## How the citations work here

Every page carries in-text citation markers like `[3]` pointing to a numbered Works Cited list at the end. Rather than printing long raw URLs in the body text, each source is a short, clean link — labeled by what it opens (**View source ↗**, **Watch video ↗**, **View PDF ↗**, **View DOI record ↗**) — so the page stays readable while the actual destination sits behind the link.
