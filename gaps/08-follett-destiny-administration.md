# Gap 8: Follett Destiny Administration Deep-Dive

*New research added for the IB Librarian Skills website, July 2026. In-text citations follow MLA 9th edition, with a bracketed number [n] alongside each for quick lookup against the numbered Works Cited list at the end.*

## 1. Destiny is a suite, not a single tool

Destiny Library Manager now sits inside a broader Library Suite alongside Destiny District Classroom Library Manager, which extends resource tracking from the central library into individual classroom collections, and Destiny Engage, which layers on personalized reading recommendations and challenges (Follett Software, "Library Suite" [1]). For an IB librarian juggling a central secondary collection plus subject-department classroom sets, the Classroom Library Manager piece is worth learning even if it wasn't part of the original rollout at your school.

An AI capability now built directly into the platform can automate parts of reporting and surface collection insights automatically, which is worth testing against your existing Power BI dashboards rather than treating the two as competing tools (Follett Software, "Destiny Library Manager" [2]). District-level dashboards added in recent releases give both librarians and administrators shared visibility into usage metrics, which is useful language to have ready for Learning Resources Committee reporting (Follett School Solutions [3]).

## 2. Titlewave and TitleWise: the collection-analysis engine

TitleWise is Follett's free collection-analysis tool, reachable directly from inside Destiny's Back Office, and it produces custom reports on collection age, subject balance, and circulation health rather than requiring a separate weeding audit built from scratch (Follett Content [4]). The workflow is: submit your MARC records to Titlewave from Destiny's Catalog menu, then run the TitleWise Analysis from your Titlewave account (Follett Community, "Learn How to Perform a TitleWise Collection Analysis" [5]).

Once a collection is uploaded, Titlewave's List tool supports sharing curated lists with colleagues, tagging "Do Not Exceed" budget notes directly on a list, and comparing a proposed list against a school's actual student population data before ordering (ESBOCES Library Guides [6]). This is the mechanism behind Titlewave's curated "IB Starter Pack" and curriculum-mapped lists referenced elsewhere in this site's core skills overview — they are built on the same tagging system.

## 3. Practical admin workflow

A short Follett-produced walkthrough shows the one-click path from Destiny into Titlewave, including how to import MARC records and refresh a TitleWise report without leaving the Destiny interface (Follett Community, "Using Titlewave with Destiny Library Manager" [7]). For day-to-day collection development, a working district library-guide recommends running the TitleWise analysis to target weeding priorities throughout the year rather than saving it for a single annual audit, and using the Tag Explorer to filter results by curriculum area before committing to a purchase (BCPS Library Information Services [8]).

## Bottom line

Three deliverables would close this gap: (1) a scheduled quarterly TitleWise re-analysis rather than a once-a-year pull, timed to align with the Term 1 CREW/MUSTIE weeding cycle already in this plan, (2) a shared Titlewave list workflow for subject-department heads requesting new titles, using the "Do Not Exceed" notes field to manage budget conversations in writing, and (3) a side-by-side test of Destiny's built-in AI reporting against the existing Power BI dashboard to see whether one can be retired.
